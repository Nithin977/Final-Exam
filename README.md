Project Overview

This android application shows the application of Room Database, RecyclerView, and various Activities with Kotlin.  

The application lets one add, read and examine user information on the local drive in Room persistence.



Features Implemented

Indenture User with Room Entity.

DAO: Insert User into database.

Load all users with Recycler View.

Open User Details by clicking on a list item.

Coroutines were used to perform data operations.

Multi-screen full functionality.



Technology Stack

Language: Kotlin

Database: Room

UI: XML Layouts

Parallelism: Kotlin Coroutines.

Architecture DAO + Entity + Database.

IDE: Android Studio
